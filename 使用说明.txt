========================================
  Arrow_Or_Arrow 使用说明（Cocos Creator 3.8.8）
========================================

一、打开工程
1. 打开 Cocos Creator 3.8.8
2. 选择「打开其他项目」，选中本文件夹：
   d:\NewGame1\Arrow_Or_Arrow
3. 等待右下角资源导入完成（resources 里会有心形/按钮/音效）

二、确认场景组件（一般已自动挂好）
【开始页】
1. 资源管理器 → assets/scene/Start
2. 双击打开，选中 Canvas
3. 检查 Inspector 是否有组件「StartMenu」
   - 若没有：点击「添加组件」→ 自定义脚本 → StartMenu

【游戏页】
1. 打开 assets/scene/Main
2. 选中 Canvas
3. 检查是否有组件「GameManager」
   - 若没有：添加组件 → GameManager

三、设置启动场景
1. 菜单：项目 → 项目设置 → 功能裁剪 旁边找到「项目设置」
2. 或：项目 → 项目设置 → 通用
3. 启动场景选 Start
（也可在编辑器里打开 Start 场景后直接点顶部「预览」）

四、预览游玩
1. 点击编辑器上方 ▶ 预览
2. 开始页点「开始游戏」进入关卡
3. 操作：
   - 单击箭头：射线判定；挡住→置灰并扣心，**仍可再点**，心扣完失败
   - 通畅：按箭头朝向**逐格爬行**移出（弯道不转向），非整块飞出
   - 底栏 ± ：缩放（内容被白色棋盘 Mask 裁剪，辅助线不跑出框）
   - 提示 / 辅助线 / 设置（打开暂停计时）
   - 返回主页/大厅：安全释放棋盘后加载 Start 场景

【重要】项目设置 → 项目数据 → 参与构建的场景 请勾选 Start 与 Main


五、已实现内容
- 浅蓝主界面（贴近参考图）+ 暖色开始页（用你提供的贴图）
- 2 个示范关（折线格子箭头）
- 3 心、计时（第1关2分钟，每3关+1分钟）
- 胜利弹窗：双倍奖励 / 下一关 / 重新开始 / 返回大厅
- 失败弹窗：增加三颗心 / 重新开始 / 返回大厅
- 「双倍奖励」「增加三颗心」目前为占位（可后续接广告）

六、若报错找不到脚本
1. 菜单：开发者 → 重新加载脚本 / 刷新
2. 确认 assets/script 下文件没有红叹号
3. 重新给 Canvas 添加 StartMenu / GameManager
